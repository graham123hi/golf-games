# Golf Games ⛳

A mobile-first app for tracking golf betting games with friends — skins, nassau, and custom games — with an AI rules assistant.

Built as a single-page web app. Installable on your phone (PWA) and works offline on the course.

## Run it locally

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Roadmap

- ✅ **Phase A** — Live online + installable app icon + offline support (PWA)
- ⏳ **Phase B** — AI rules assistant (Claude API via a serverless function)
- ⏳ **Phase C** — Live shared games: real accounts + join-by-code + realtime score sync (Supabase)
